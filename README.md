# Qrar Login

Modern luxury login page for Qrar SaaS built with React and Vite.

## Features
- Modern UI/UX design
- Responsive layout
- Brand colors integration
- Smooth animations

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
